# StopWatch

A simple gym stopwatch for iOS, optimized for rest periods between sets.

## Features

- **One-tap restart** - Reset to zero and immediately start again
- **Lock Screen Live Activity** - See elapsed time and restart without unlocking
- **Dynamic Island support** - Quick glance at your timer
- **Minimal UI** - Just start, stop, and restart

## Requirements

- iOS 17.0+
- Xcode 15.0+

## Building

1. Open `StopWatch.xcodeproj` in Xcode
2. Select your development team in Signing & Capabilities
3. Build and run
